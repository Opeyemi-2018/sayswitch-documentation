## Retrieve Customers

This endpoint retrieves the complete list of customers registered in the system.

### Endpoint  
`GET https://backendapi.sayswitchgroup.com/api/v1/customer`

### Request Headers

| Header        | Type   | Description                          |
|---------------|--------|------------------------------------|
| Authorization | string | Bearer token containing `YOUR_SECRET_KEY` |

---

### Example Request

```js copy
GET https://backendapi.sayswitchgroup.com/api/v1/customer
```

### Example cURL Command
```js
curl -X GET https://backendapi.sayswitchgroup.com/api/v1/customer 
  -H "Authorization: Bearer YOUR_SECRET_KEY"
```
