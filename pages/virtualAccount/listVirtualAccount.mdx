## List Virtual Accounts

This endpoint retrieves a list of all virtual bank accounts that have been created on your platform.

#### API Endpoint

`GET https://backendapi.sayswitchgroup.com/api/v1/list_dedicated_accounts`

### Request Headers

| Header          | Type     | Description                                            |
| --------------- | -------- | ------------------------------------------------------ |
| `Authorization` | `string` | Your API secret key, prefixed with `Bearer `. Example: `Bearer ECRET_KEY`. |

### Request Body

This endpoint does not require a request body.

### Sample Request

```bash
curl -X GET https://backendapi.sayswitchgroup.com/api/v1/list_dedicated_accounts 
  -H "Authorization: Bearer SECRET_KEY"
```
