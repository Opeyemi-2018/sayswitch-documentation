## List Payouts

The List Payout Endpoint Retrieves a list of all payouts made from your account.

### Endpoint 
`GET https://backendapi.sayswitchgroup.com/api/v1/payout`

### cURL Request

```json copy
curl -X GET "https://backendapi.sayswitchgroup.com/api/v1/payout \
  -H "Authorization: Bearer SECRET_KEY" \
  -H "Content-Type: application/json"
```

### Response Explanation


| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Indicates if the API request was successful |
| `message` | string | Human-readable status message |
| `data` | array | Contains array of payout objects |
| `id` | integer | Unique payout identifier |
| `reference` | string | Payout reference number |
| `sessionid` | string | Transaction session ID |
| `currency` | string | Currency code (NGN) |
| `amount` | string | Payout amount in major units |
| `fee` | string | Transaction fee amount |
| `bank_code` | string | Bank institution code |
| `bank_name` | string | Beneficiary bank name |
| `account_number` | string | Recipient account number |
| `account_name` | string | Recipient account name |
| `countryCode` | string | Country code (NG) |
| `serviceCode` | string/null | Service provider code |
| `paymentMode` | string | Payment method (bank) |
| `narration` | string | Transaction description |
| `sender` | string | Sender name |
| `domain` | string | Environment (live/test) |
| `requestIp` | string/null | Initiator IP address |
| `initiator` | string/null | Who initiated the payout |
| `status` | string | Payout status (success/pending/failed/reversed) |
| `updated_by` | string/null | Who last updated the record |
| `updated_on` | string/null | Last update timestamp |
| `beneficiary` | integer | Beneficiary ID (0 if none) |
| `subaccount` | object/null | Subaccount details |
| `gateway` | string/null | Payment gateway used |
| `batch_id` | string/null | Batch reference ID |
| `created_at` | string | Creation timestamp (ISO 8601) |
| `updated_at` | string | Last update timestamp (ISO 8601) |
